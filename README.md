# Coach You React - Mentorías profesionales

Reserva tus sesiones de mentoría uno a uno con profesionales altamente calificados

[![Screen](./screen.png)](https://coach-you-react.vercel.app/)

### Instalación del proyecto

```bash
npm install

npm run dev
```
